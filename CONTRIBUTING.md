# Guía de Contribución para Daily Notes

¡Qué bueno que quieras ayudar a mejorar esta herramienta!

## ¿Cómo puedes ayudar?

1. **Nuevas Funciones**: Como búsqueda de texto dentro de las notas o soporte para carpetas.
2. **Documentación**: Mejorar los ejemplos de prompts o corregir errores en el README.
3. **Bug Hunting**: Reportar cualquier error en el manejo de archivos en diferentes sistemas operativos.

## Pasos para colaborar

1. Haz un Fork.
2. Crea tu rama: `git checkout -b feature/mejora-increible`.
3. Haz tus cambios y asegúrate de que `server.py` siga corriendo.
4. Envía tu Pull Request a la rama `main`.

¡Gracias por ser parte de la comunidad!
